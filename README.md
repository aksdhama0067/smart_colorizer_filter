# AI Smart Filter & Deep Learning Image Colorizer 🎨

An advanced computer vision application built with Python, OpenCV, and NumPy that leverages Deep Learning convolutional neural networks (CNN) to automatically transform grayscale images into vibrant, realistically colored photos. Beyond colorization, the system includes a robust matrix filtering suite designed for digital image processing and structural enhancements.

This repository is optimized for modern development environments, featuring complete cross-platform path resolution and an automated recovery sub-system to manage neural network dependency downloads seamlessly.

---

## 🔥 Key Capabilities & Real-World Examples

This application functions as a multi-purpose image processing pipeline. It reads raw pixel data, maps it across targeted transformation matrices, and outputs high-resolution graphics.

### 1. Deep Learning Image Colorization
The core feature utilizes a Caffe-based deep learning model trained on millions of color images. It treats colorization as a classification task, mapping grayscale luminance values ($L$) to predicted chrominance channels ($a$ and $b$) in the Lab color space. This makes it perfect for:
* **Historical Photo Restoration:** Bringing antique family heirlooms, vintage portraits, and mid-20th-century street photography back to life.
* **CCTV / Forensic Cleanup:** Enhancing monochrome security footage or low-light captures for better contextual analysis.
* **Artistic Colorization:** Experimenting with stylized black-and-white photography to generate unique aesthetic contrasts.

#### Example Command:
```bash
python smart_filter.py -i vintage_photo.jpg --colorize -o restored_vibrant.jpg
